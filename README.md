## Project Description
This project focuses on the core building blocks of JavaScript. I developed a logic-based calculator and a student grade tracking system to practice handling variables, data types, loops, and functional logic. The goal was to understand how to manipulate data and solve problems using a programmatic approach.

## Technologies Used
- HTML5
- JavaScript (ES6+)
- Git & GitHub
- Markdown

## Features
- **Functional Calculator:** Supports addition, subtraction, multiplication, division, modulus, and exponentiation.
- **Error Handling:** Includes logic to catch and handle division by zero.
- **Grade Management System:** An object-oriented tool that calculates student averages and identifies high-performing vs. struggling students.
- **Data Transformation:** Extensive use of modern array methods like `.map()`, `.filter()`, and `.reduce()`.

## How to Run
1. Clone this repository: `git clone https://github.com/s7085751-afk/iyf-s10-week-04-s7085751-afk.git`
2. Open `index.html` in your browser.
3. Open the **Browser Console** (F12 or Right-click > Inspect > Console) to see the output and interact with the functions.

## Lessons Learned
While building this project, I learned the importance of "Dry" (Don't Repeat Yourself) code. Instead of writing long scripts, I broke the logic down into small, reusable functions. I also gained a deep understanding of how objects and arrays work together to store and organize complex information.

## Challenges Faced
The biggest challenge was realizing I had missed a core requirement (the interactive calculator) after I thought I was finished. I solved this by pulling my code back from GitHub, implementing the missing logic using a `switch` statement, and re-testing all operations in the console to ensure they worked before re-submitting.
