# IYF Season 10 - Week 1: Web Foundations

## Project Overview
This repository contains my first multi-page portfolio website, built as part of the IYF Weekend Academy. The project focuses on using **Semantic HTML5** to create an accessible and well-structured user experience.

## Student Information
- **Name:** Sharleen Salma
- **Track:** Web Development
- **Institution:** Kenyatta University

## Features
- **Multi-page Navigation:** Linked Home, About, Projects, and Contact pages.
- **Semantic Structure:** Utilization of `<header>`, `<nav>`, `<main>`, `<section>`, and `<article>` tags.
- **Form Validation:** A contact form with fieldsets, legends, and required input attributes.
- **Mobile Responsive:** Basic meta-tag setup for various screen sizes.

## Technologies Used
- HTML5
- Semantic Markup
- GitHub Pages for Deployment
